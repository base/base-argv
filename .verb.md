## Usage

```js
var Base = require('base-methods');
var argv = require('{%= name %}');

var app = new Base();
app.use(argv());

// parse argv
var args = app.argv(['foo', 'bar', '--set=a:b']);
console.log(args);
```

Results in:

```js
{ set: { a: 'b' }, tasks: [ 'foo', 'bar' ] }
```

This object can then be passed to something else for further processing.
